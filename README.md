# Mercurials

## Development
1. Start local test blockchain RPC node on http://localhost:8545 
  ```
  $ anvil
  ```
1. Watch Solidity contract files and automatically deploy them when modified and saved
  ```
  $ npm run watch
  ```
1. Start the Next.js server
  ```
  $ npm run dev
  ```
